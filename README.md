# 🏏 IPL Data Analysis Dashboard

## 📌 Project Overview

This project analyzes IPL match data using Python and Power BI. The dashboard provides insights into total matches, runs scored, wins, team performance, venue performance, and match details.

---

## 🛠️ Tools Used

- Python
- Pandas
- Power BI
- CSV Dataset
- Git & GitHub

---

## 📊 Dashboard Features

### KPI Cards
- Total Matches
- Total Runs
- Total Wins

### Visualizations
- Runs by Team
- Runs by Venue
- Matches by Team
- IPL Match Details Table

### Interactive Filters
- Team Slicer
- Venue Slicer

---

## 📂 Project Files

- `ipl_data.csv` - Dataset
- `ipl_analysis.py` - Python Analysis Script
- `dashboard.png` - Dashboard Screenshot
